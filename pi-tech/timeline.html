20 Major Events (1981–2024)

1981
IBM PC Launches — Software Liability Era Begins
IBM PC (August 1981) creates mass-market software industry. Courts begin grappling with software liability — product vs service distinction. First US technology E&O policies written by St Paul Companies. VER
1988
Morris Worm — First Major Internet Incident
Robert Morris worm (November 1988) infects ~6,000 Unix computers — first widely publicised internet security incident. Highlights connectivity risk. Early catalyst for what will become cyber/tech E&O convergence. VER
1994
Intel Pentium FDIV Bug
Intel Pentium processor floating-point division bug discovered. Recall cost ~$475m. Paradigm example of hardware/software defect causing systemic financial loss. Establishes principle: technology defects = quantifiable economic claims. VER
1998
NHS/Government IT Outsourcing Wave
UK government begins large-scale IT outsourcing contracts (NHS, HMRC, DWP). EDS, Accenture, CSC win billion-pound deals. Seeds of future NHS NPfIT and HMRC IT failures — largest tech E&O exposures in UK history. EST
1999
Y2K — $600bn Remediation, ~$1bn Claims
Global Y2K remediation programme: $600bn+ spent worldwide. Insurance claims ~$1bn — far below feared losses. Lloyd's tech PI hard market 1999–2001. Y2K exclusions become standard on new policies. VER
2000
Dot-Com Bust — IT Project Claims Wave
Dot-com collapse March 2000 triggers wave of failed IT project claims. Hundreds of e-commerce implementations abandoned mid-build. Tech E&O market faces first significant claims cycle. VER
2003
NHS National Programme for IT Launched
NHS NPfIT — £12.4bn contract for national electronic patient record system. Largest civilian IT project in history. Ultimate failure (2011) with £9.8bn wasted — major tech E&O and public contract PI exposure. VER
2006
AWS Launch — Cloud E&O Era Begins
Amazon Web Services launches (March 2006). Cloud computing transforms tech liability — from discrete project delivery to continuous service provision. Tech E&O evolves to cover ongoing SaaS/cloud service failures. VER
2011
NHS NPfIT Abandoned — £9.8bn Wasted
NHS National Programme for IT formally abandoned. £9.8bn of taxpayer money spent with no functioning national EPR. Largest public-sector IT failure — tech E&O / government contract PI claims; confidential settlements. VER
2012
Knight Capital — $440m Algorithmic Trading Loss
Knight Capital Group: defective market-making algorithm deployed in live environment (August 2012). 45 minutes of erroneous trades = $440m loss. Company sold within days. Paradigm tech E&O event — single software defect causes corporate-level financial loss. VER
2016
SWIFT Banking System Fraud — Bangladesh Bank
Bangladesh Bank: $81m fraudulently transferred via SWIFT network. Exploits in SWIFT messaging software and bank security. Tech E&O/cyber interface debate: was it software vulnerability (F4) or security failure (CY)? VER
2017
NotPetya — $10bn Losses, War Exclusion Dispute
NotPetya cyberattack (June 2017): disguised as ransomware but destructive. Maersk, Merck, FedEx among victims; $10bn+ economic damage. Insurers invoke war exclusion. Merck wins $1.4bn insurance case 2023. Tests cyber/tech E&O boundary. VER
2018
TSB Banking IT Migration Failure — £330m
TSB migrates from Lloyds Banking Group platform to Proteo4UK (Sabadell). April 2018: migration fails catastrophically — 1.9m customers locked out; online banking down for weeks. £330m compensation, FCA £48.6m fine. IBM consulting firm Sabis implicated. VER
2018
GDPR — Data Processing E&O Claims Emerge
EU GDPR (May 2018). Technology firms advising on data architecture face PI claims for GDPR non-compliance advice. Data breach notifications surge — testing tech E&O/cyber boundary on data loss caused by misconfiguration vs attack. VER
2020
SolarWinds Supply Chain Attack
SolarWinds Orion software update compromised by Russian state actor. 18,000+ organisations (including US Treasury, DHS) receive malicious update. Tech E&O/cyber crossover — compromise of legitimate software update = supply chain PI risk. VER
2021
Kaseya VSA Attack — MSP Supply Chain
Kaseya VSA (IT management software for MSPs) compromised by REvil ransomware group. 1,500 businesses downstream via ~60 MSPs. Tests whether MSP E&O policy responds to upstream vendor compromise. Tech E&O market adds MSP supply-chain exclusions. VER
2021
Log4Shell — Emergency Patching Costs
Log4j vulnerability (CVE-2021-44228, Dec 2021) — critical flaw in widely used Apache logging library. Billions of devices affected. Emergency patching costs borne by thousands of tech firms. Tech E&O PI debate: is patching cost a covered loss if client's system was compromised via Log4Shell? VER
2024
AT&T Network Outage — Emergency Services Disruption
AT&T network outage (February 2024): 8-hour outage affecting ~70,000 emergency calls. FCC investigation; $9m penalty agreed. Telecom PI (F5) paradigm: network failure → emergency services disruption → regulatory/PI exposure. VER
2024
CrowdStrike Falcon Update — 8.5 Million Devices Crashed
CrowdStrike defective Falcon sensor content update (19 July 2024) crashes 8.5 million Windows devices globally. Airlines, hospitals, broadcasters, banks grounded. Delta Air Lines: $500m loss + lawsuit against CrowdStrike. Estimated $10bn+ total economic loss; insured ~$800m–1bn. Tech E&O paradigm event of the decade. VER
2024
EU AI Act — AI Liability Framework
EU AI Act (2024) establishes risk-based regulatory framework for AI systems. High-risk AI systems face mandatory conformity assessments, documentation and oversight. Tech providers deploying AI in critical systems (healthcare, employment, credit) face new PI exposure. AI hallucination liability emerging in legal and financial contexts. VER
